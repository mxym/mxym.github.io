## jadx-mcp

codex配置jadx-mcp时不能和ida-pro-mcp一样，如这篇文章所述，在别的地方没什么问题就codex不行（

[codex 安装 JADX-AI-MCP 报错处理 - 开发调优 - LINUX DO](https://linux.do/t/topic/1508247)

需要这样

```
[mcp_servers.jadx-mcp]
type = "steamable-http"
startupTimeout = 30000
toolTimeout = 300000
url = "http://127.0.0.1:8651/mcp"
```

然后得手动启动

```
uv run .\jadx_mcp_server.py --http --port 8651 --jadx-port 8650
```

## opencode

登录openai时可能出问题报403

魔法的问题

如这篇文章所述[Windows 下 OpenCode使用 ChatGPT Pro/Plus 鉴权报错 403 的排查与解决_chatgpt 403-CSDN博客](https://blog.csdn.net/m0_63460124/article/details/157400240)

总结是看魔法端口改三个环境变量，我是手动改的，端口改魔法端口

```
$env:HTTPS_PROXY = "http://127.0.0.1:7890"
$env:HTTP_PROXY  = "http://127.0.0.1:7890"
$env:NO_PROXY    = "localhost,127.0.0.1,::1"
```

然后就可以登录了
